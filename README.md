## Usage
### via NPM
#### With installation
```shell
npm i -g emusk
```
```sh
emusk <option>
``` 
#### No installation
```sh
npx emusk <option>
```
### via Docker

## Configuration file
```
emusk --config-file=/config/emusk.json <option>
```
default `./.emuskrc`
